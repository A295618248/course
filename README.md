# 星河艺术中心小程序

一个面向艺术培训机构的微信小程序演示项目，包含课程展示、活动推广、报名登记和报名记录查看等核心能力，适合少儿美术、书法、陶艺、综合创意类培训业务快速上线展示。

## 已实现功能

- 首页总览
  - 机构定位与招生亮点
  - 课程、活动、累计咨询数据展示
  - 推荐课程与近期活动入口
- 课程中心
  - 按课程分类筛选
  - 展示适龄、课时、班型、价格与课程亮点
  - 课程详情页支持直接报名
- 活动中心
  - 展示展览、开放日、体验营等活动
  - 支持按状态筛选
  - 活动详情页可查看流程安排并直接报名
- 报名登记
  - 支持课程报名、活动报名和通用咨询
  - 包含手机号、年龄、兴趣方向等基本校验
- 报名中心
  - 本地存储报名记录
  - 查看课程咨询、活动报名统计
  - 支持清空演示数据

## 使用方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择当前仓库根目录
4. AppID 可先使用测试号或无 AppID 模式进行预览
5. 导入后即可直接运行

## 说明

- 当前为前端演示版本，课程、活动数据来自 `utils/data.js`
- 报名记录保存在微信本地存储中，便于演示完整流程
